# Repo Onboarding Skill Template

Use this as a portable skill template when a tool supports `SKILL.md`-style workflows.

Start with `docs/ai/CONTEXT_INDEX.md`. For localized structures, use the localized equivalent such as `docs/ki/KONTEXT_INDEX.md` or the path from the selected File-Map.

Repository evidence wins over stale documentation. Load only task-relevant files, especially `TECH_STACK.md`, `BUILD_AND_TEST.md`, `DEPENDENCIES.md`, `RUNTIME_ENVIRONMENT.md`, `SECURITY_RULES.md` and `REVIEW_CHECKLIST.md`.

Do not invent undocumented behavior or claim vendor support.
