# Tool Entrypoint Templates

These templates are short copy targets for agent tools and IDE assistants.

Use them only when the target repository actually uses the matching tool. They point to `docs/ai/CONTEXT_INDEX.md` or a localized equivalent instead of copying the whole AI knowledge base.

For localized output, resolve the equivalent files through `i18n/file-map.<code>.yml`. German examples use `docs/ki/KONTEXT_INDEX.md`.

These files are compatibility templates, not vendor endorsements or support guarantees.
