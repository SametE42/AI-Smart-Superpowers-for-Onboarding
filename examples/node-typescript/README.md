# Node/TypeScript Repository Example

This example describes how to apply the AI onboarding standard to a Node.js or TypeScript project.

## Suggested AI documentation

```text
docs/ai/
├─ MASTER_SYSTEM.md
├─ ARCHITECTURE.md
├─ PROJECT_MEMORY.md
├─ STYLE_GUIDE.md
├─ REVIEW_CHECKLIST.md
├─ SECURITY_RULES.md
└─ ONBOARDING.md
```

## Recommended focus areas

- package manager and runtime version
- build, lint and test commands
- framework conventions
- API boundaries
- environment variables and secrets handling
- generated files and ignored folders

## Example validation commands

```text
npm install
npm run lint
npm test
npm run build
```

Adjust commands to the actual repository before documenting them as authoritative.
