# Legacy Codebase Example

## Use Case

Older repository with fragile behavior, missing tests and undocumented decisions.

## Repository-Typ

Legacy application or library.

## verwendeter Modus

Standard Integration.

## verwendete Sprache

English.

## verwendeter Strukturmodus

Canonical.

## erkannter oder angenommener Stack

Unknown until repository evidence is inspected.

## verwendete Templates

`ERROR_PATTERNS.md`, `EVIDENCE_MAP.md`, `PROJECT_MEMORY.md`, `DECISIONS.md`, `FRESHNESS.md`.

## Agent zuerst lesen

1. `docs/ai/EVIDENCE_MAP.md`
2. `docs/ai/ERROR_PATTERNS.md`
3. `docs/ai/PROJECT_MEMORY.md`

## Human Review

Review fragile areas and unknowns before refactoring.

## Legacy Signal

Use `[UNKNOWN]` and `[ASSUMPTION: ...]` instead of inventing missing architecture history.
