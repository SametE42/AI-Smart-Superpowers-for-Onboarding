# Skills

This directory contains portable workflow skills for the onboarding framework itself.

Skills are optional task workflows. They keep repeatable agent behavior compact and point back to `docs/ai/` or the localized equivalent instead of duplicating the full knowledge base.

Available skills:

| Skill | Purpose |
|---|---|
| `repo-onboarding` | Prepare a repository knowledge base before implementation work. |
| `docs-ai-review` | Review an existing AI knowledge base against repository evidence. |
| `docs-ai-update` | Update durable AI documentation after approved repository changes. |

These files are tool-neutral templates and do not claim vendor support.
