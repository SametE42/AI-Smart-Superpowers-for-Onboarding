# Agent Workflows

This folder contains short, tool-neutral workflow files for AI coding agents.

Use these workflows before planning, implementation, testing or review when a repository needs evidence-first onboarding and a reviewed `docs/ai/` knowledge base.

| Workflow | Use when |
|---|---|
| [`01-repository-onboarding.md`](01-repository-onboarding.md) | Starting work in an unfamiliar, stale or high-risk repository. |
| [`02-evidence-first-analysis.md`](02-evidence-first-analysis.md) | Gathering facts, unknowns and assumptions from repository evidence. |
| [`03-ai-knowledge-base.md`](03-ai-knowledge-base.md) | Creating or updating the target repository's `docs/ai/` files. |
| [`04-pre-coding-check.md`](04-pre-coding-check.md) | Checking whether coding, testing, planning or review can safely begin. |
| [`05-handover-and-memory-refresh.md`](05-handover-and-memory-refresh.md) | Refreshing handover notes and project memory after significant changes. |

These are structured agent workflows. They do not imply official integration, endorsement or compatibility certification with any specific AI coding tool.
