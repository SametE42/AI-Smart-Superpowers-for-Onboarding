# Configuration

This folder contains machine-readable repository standards.

`standard-docs.yml` is the source of truth for:

- the Conceptual Core document list used for public explanation;
- Minimal, Standard and Enterprise installer document lists;
- validation that installer modes, language file maps and documentation examples remain aligned.

`external-link-allowlist.txt` is intentionally empty by default. Add a URL
pattern only when a public link cannot be checked reliably by the optional
network link-check workflow and the limitation is documented.

Keep this file reusable. Do not add project-specific progress notes, secrets or local machine data here.
